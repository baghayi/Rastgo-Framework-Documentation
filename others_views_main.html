بخش های دیگر

این پوشه هم برای قرار دادن قالب ها مون هست و فایل هایی که باید توسط کلاس قالبمون رندر بشه

برای اینکه بشه برای سیستم فالب های متنوع ای درست کرد و هر موقع خواسته شد بدون مشکل بشه تغییرشون داد / بجای اینکه همه فایل ها تو همین پوشه View قرار بگیرن طوری درست کردم که هر قالب یه پوشه و نام خود رو داشته باشه.

که اسم قالب پیش فرض این فریمورک defaultTemplate هستش

در مورد فایل ها جاوااسکریپت یا css یاعکس و ... که کجا باید قرار بگیرن و چطوری لود بشن و آدرسشون چی خواهد بود در این لینک زیر توضیح داده شده است:
views (محل، فایل های مورد نیاز قالب)

فایلی برای مثال در مورد نحوه عملکرد بخش View بنام index.php و داخل defaultTemplate هستش که یه ساختار html ای داره و چند تا متغیر توشون تعریف شده برای (تست) و رندر شدن

که در مورد رندر شدن و نحوه قرار دادن متغیر ها و ... در لینک زیر توضیحاتی داده شده:
Template

توجه کنید که می تونید داخل فایل های قالب مثل index.php در پوشه defaultTemplate دقیقا مثل php عمل کرده و از foreach و ... استفاده کنید و مشکلی ایجاد نخواهد شد (که بستگی به نحوه رندر کردن کلاس تون داره که برای این فریمورک نحوه رندر کردنش توضیح داده شده و اگه با کلاس خودتون جایگزین کنید نسبت به کار کلاس خودتون این هم باید تغییر پیدا کنه طبیعتا!)
